Informació del curs

Aquest programa formatiu t'introduirà de ple al disseny digital. Durant les classes pràctiques aprendreu l'estructura lògica amb HTML, l'estilitzat avançat de components multimèdia amb CSS3 i l'arquitectura Responsive Web Design per garantir un funcionament impecable en dispositius mòbils, tablets i ordinadors de sobretaula.

UNITAT FORMATIVA 1: Estructuració de Continguts en HTML

Aprèn lús correcte de les etiquetes semàntiques fonamentals, jerarquies de text i optimització SEO estructural en cercadors.

UNITAT FORMATIVA 2: Estils Gràfics i Maquetació amb CSS

Domina Flexbox, CSS Grid Layout, transicions fluides de color, selectors avançats i herències decoratives de disseny.

UNITAT FORMATIVA 3: Disseny de Llocs Webs
